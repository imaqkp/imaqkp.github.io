# 毕业前夜 · 时光重启

这是一个可直接运行、可部署的校园回忆互动网页小游戏。

## 文件说明

- `index.html`：网页结构
- `style.css`：视觉样式
- `script.js`：游戏逻辑

## 本地运行

直接双击 `index.html` 即可打开。

## 部署到 Netlify

1. 登录 Netlify
2. 新建站点
3. 选择手动部署 / drag and drop
4. 把整个 `campus_restart_game` 文件夹拖进去
5. 等待生成网址

## 可修改内容

- 修改 `index.html` 里的标题、章节文案
- 修改 `script.js` 里的题目、选项、分数、结局
- 修改 `style.css` 里的颜色、字体、卡片样式
